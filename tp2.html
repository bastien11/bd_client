Application Web:

Introduction:
Dans ce cour, nous avons pu voir différentes Arichtecture cliente ainsi que des application cliente Web,
mais aussi des application serveur web.

Model Vue Controller:

C'est une des architecture cliente les plus connus et utilisées avec le MVP (Model Presenter View).
Ce qui permet de structurer son application en séparant le model de la vue avec le controller pour gérer tout ça.

Dans une application cliente web, plusieurs solutions sont possible pour gérer la persistance des données.

  • Limiter à la fenêtre : Session d'enregistrement, sessions storage.

  • Limiter au navigateur : Enregristrement locale, Local storage.

  • Cookies / Cache Storage: Permet de stocker quelques ko.

  • Web SQL: Donnée structurés disponible sur WebKit, Opera.

  • etc...

Il existe beaucoup de framework et langage pour créer et déployer une application web. Tel que Symphony, Django pour les framwork et PHP, NodeJS pour les langages.
